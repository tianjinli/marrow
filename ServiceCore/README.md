# ServiceCore 模块

ServiceCore 是系统中的 **公共辅助类库模块**，用于给其他模块提供核心功能。

## ✨ 功能简介
- 命令行参数解析工具（原先的已废弃）
- UTF8 字符转换工具（来自 Cocos）
- INI 文件读写工具（来自 Github）
- 字符串辅助工具（自己封装）
- Base64 工具（暂未使用）
- 数据锁（类似 std::lock_guard 已废弃）
- 数据队列（已废弃）
- 加密组件（仅 INI 文件用）
- 配置数据（INI 进一步封装）
- MD5 工具（暂未使用）
- 系统服务（常用信息获取）
- SHA1 工具（暂未使用）
- 线程/线程池 （前者类似 std::thread 已废弃）
